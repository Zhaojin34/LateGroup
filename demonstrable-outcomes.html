Demonstrable Outcomes

Project Motivation

There are several web and mobile applications assisting people in scheduling their events, however, most of the former, and latter are aimed at a strictly professional lifestyle. Our application will use a combination of event planning, and general day-to-day management. The panda is to provide positive reinforcement – with the goal to ultimately support users in developing and maintaining good habits, to create a good balance between professional and personal events.

The group knows how to code functions locally but have not built/run a web application prior to this. It is anticipated that the group will need to learn basic to moderate database/server principles.


Minimum Viable Features

We will produce a login and sign up system.

We will produce a page for users to set up their tasks.

We will produce a Dashboard showing all tasks built by the user.

We will produce a calendar and show users’ tasks on it.

We will produce a page for users to interact with their panda.

Ensure the user is able to:

1. Open a sign-up page

2. Open a login page

3. Input username

4. Check username is available

5. Input password

6. Confirm password

Ensure the user is able to:

1. Add a new task

2. Input the name of task

3. Choose the type of task

4. Input the description of task

5. Choose the repeat cycle of task

Ensure the user is able to:

1. See his tasks list by types

2. Open a task and view details

3. Rearrange tasks

4. Edit a task

5. Delete a task

Ensure the user is able to:

1. Open the calendar

2. Change the type of calendar

3. Check all tasks among the period

4. Open a task and view details in calendar

Ensure the user is able to:

1. View picture of panda

2. View attributes of panda

3. Check his supplies and gold

4. Interact with panda

Extended Viable Features

We will produce a store page and shows commodity.

We will produce a message board and users can communicate on it.

We will produce a ranking page for pandas.

Ensure the user is able to:

1. Check number of gold

2. Check the information of items

3. Reflash the store once a day

Ensure the user is able to:

1. View other users' messages

2. Create a new message

3. Leave replay to a message

4. Delete his message and replay

5. Reflash message board and view the latest messages

Ensure the user is able to:

1. View the top ranking board

2. Check his rank
